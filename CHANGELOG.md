# Changelog

All notable changes to this project will be documented in this file.

### Added
- Initial project setup and core architecture
- Blueprint graph data structures
- Verse digest file parsing system
- Workspace file loading functionality
- Variable management with properties
- Code generation to valid `.verse` files

- File persistence (blueprint save/load)
- GitHub Actions CI/CD pipeline

- UEFN plugin integration
